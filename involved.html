Get Involved

Boro International is a non-profit event used to educate our society about the different cultures that influence it.
We welcome those who are interested in contributing to this event via, volunteer, vendor or entertainer.
Below are links to the different application forms. A brief discription and Contact link is located here as well to help you get involved.
Please not that the application will be considered by the event officials

VENDOR

Are you interested in selling merchandise, food, or even having your own exhibit? if so please click the link below where you will be
redirected to the application page.

Vendor Application


ENTERTAINER

Entertainment is an important part of this event, as we hope to use different culutral performances to educate our community.
If you have a unique dance, song, or any performance you are interested in sharing with the community please click the link below
where you will be directed to the entertainment application page.
PLEASE NOTE THAT EACH PERFORMANCE IS SCHEDULED IN 15 MINUTE INTERVALS (THIS INCLUDES THE SETUP PROCESS)

interested in showcasing. However you still would like to get involved. We are always looking for volunteers. This application form
which is located by clicking the link below allows you to apply for a volunteer position. This includes parking assistance, cleanup
stage management, etc.
